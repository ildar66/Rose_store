• Краткое описание.

              Работа со справочником «Справочник Банков».

  • Предусловия(pre-conditions).

              Идентификация клиента.

  • Основной поток событий.

1.        Вариант использования начинается когда клиент выбирает ссылку «Справочники».

2.        Система предлагает список справочников:

a)        Личные справочники.

b)        Общие справочники.

3.        Клиент выбирает «Справочник Банков» из «Общие справочники».

4.        Система предлагает список банков. Если во время запроса возникают ошибки выполняеться поток ошибок Е1.

5.        Вариант использования завершается.

 

  • Поток ошибок.

              Е1:ошибка при выводе списка корреспондентов:

1.        Система сообщает клиенту об ошибке и дает номер телефона службы поддержки банка клиента.

2.        Вариант использования завершаеться.
